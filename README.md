# Go Rabbitmq Example
This is rabbitmq queue example using go language.

<img src="https://github.com/susimsek/go-rabbitmq/blob/master/images/go-rabbitmq.png" alt="Go Rabbitmq Example" width="50%" height="50%"/> 

## Prerequisites

* Go 1.14.x
* Docker 19.03.x
* Docker Compose 1.25.x

## Installation

```sh
docker-compose up -d
```

## Used Technologies

* Amqp
